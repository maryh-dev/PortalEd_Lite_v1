# PortalEd_Lite

## Descrição
O PortalEd_Lite é uma plataforma educacional desenvolvida para facilitar o acesso ao aprendizado online de forma simples, organizada e intuitiva. O sistema permite que alunos visualizem cursos, acompanhem seu progresso e realizem seus estudos em um ambiente digital acessível.

Esta versão Lite possui funcionalidades essenciais para gerenciamento de cursos e aprendizagem, servindo como base para futuras expansões do projeto.

## Objetivo
Desenvolver uma plataforma educacional que permita aos usuários:

- Realizar cadastro e login
- Recuperar senha
- Visualizar cursos disponíveis
- Matricular-se em cursos
- Acompanhar o progresso de aprendizagem
- Favoritar cursos
- Assistir aulas
- Receber notificações do sistema
- Gerenciar configurações da conta

## Regras de Negócio

1. O aluno deve possuir cadastro para acessar o sistema.
2. O e-mail deve ser único para cada usuário.
3. Apenas administradores podem cadastrar, editar ou excluir cursos.
4. Um aluno pode se matricular em vários cursos.
5. O progresso do aluno é atualizado conforme a conclusão das aulas.
6. Cursos favoritados ficam disponíveis na área de favoritos.
7. O sistema registra ações importantes para auditoria.
8. O administrador não pode ser criado pelo sistema; a conta é cadastrada previamente.

## Fluxo Básico de Utilização

1. O usuário realiza cadastro.
2. Efetua login.
3. Visualiza os cursos disponíveis.
4. Realiza matrícula em um curso.
5. Acessa as aulas.
6. Acompanha seu progresso.
7. Conclui o curso.
